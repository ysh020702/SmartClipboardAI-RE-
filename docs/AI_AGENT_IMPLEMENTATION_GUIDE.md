# AI Agent 구현 가이드

이 문서는 이전 Single-Turn `TopicAgent` 구현 가이드의 자리 표시자입니다.

현재 SmartClipboardAI의 AI Agent 구현 기준은 아래 문서 하나로 둡니다.

- `docs/AI_AGENT_MULTI_RETRIEVAL_ANALYSIS.md`

## 사용 규칙

- 새 구현 작업자는 이 문서를 구현 지시서로 사용하지 않습니다.
- Planner, Retriever, Action Drafter, Tool Executor, Cluster Agent 설계와 단계별 구현 순서는 `AI_AGENT_MULTI_RETRIEVAL_ANALYSIS.md`를 따릅니다.
- 과거 Gemini MVVM 재배치, Single-Turn 분석, 임시 UX 시뮬레이션 문맥은 현재 구현 기준이 아닙니다.
